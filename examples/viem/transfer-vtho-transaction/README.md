# Transfer VTHO Example

[![Open in CodeSandbox](https://codesandbox.io/static/img/play-codesandbox.svg)](https://codesandbox.io/s/github/vechain/vechain-sdk-js/tree/sdk-v3/examples/thor/transfer-vtho-transaction)

## Usage

```bash
yarn dev
```

or

```bash
tsx index.ts
```

## What it demonstrates

* Creating a private key signer
* Creating a transfter VTHO clause using the ClauseBuilder
* Executing the transaction, which includes estimating gas, signing and sending to Thor
* Waiting for the transaction receipt
