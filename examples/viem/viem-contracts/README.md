# Viem Contracts Example

This quick-start example shows the simplest and most common way to interact with smart contracts on VeChain using the Viem-based API. It walks through creating a PublicClient, defining a minimal ERC-20 ABI, loading a contract instance, and performing basic read-only calls such as balanceOf, symbol, and decimals.
