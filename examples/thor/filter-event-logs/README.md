# Filter Event Logs

[![Open in CodeSandbox](https://codesandbox.io/static/img/play-codesandbox.svg)](https://codesandbox.io/s/github/vechain/vechain-sdk-js/tree/sdk-v3/examples/thor/filter-event-logs)

## Usage

```bash
yarn dev
```

or

```bash
tsx index.ts
```

## What it demonstrates

* Creating an ABIEvent from an event signature string
* Encoding topics to be used for event filtering
* Filtering raw event logs from Thor
* Decoding raw event logs
