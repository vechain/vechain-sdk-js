# Build Transaction Request

[![Open in CodeSandbox](https://codesandbox.io/static/img/play-codesandbox.svg)](https://codesandbox.io/s/github/vechain/vechain-sdk-js/tree/sdk-v3/examples/thor/build-transaction-request)

## Usage

```bash
yarn dev
```

or

```bash
tsx index.ts
```

## What it demonstrates

* How to use the Transaction Builder to build a TransactionRequest
* How to use TransactionBodyOptions to build TransactionRequest
* How to sign and send the TransactionRequest to Thor
* How to wait for the transaction receipt
