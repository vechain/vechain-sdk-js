# Filter VET Transfers

[![Open in CodeSandbox](https://codesandbox.io/static/img/play-codesandbox.svg)](https://codesandbox.io/s/github/vechain/vechain-sdk-js/tree/sdk-v3/examples/thor/filter-vet-transfers)

## Usage

```bash
yarn dev
```

or

```bash
tsx index.ts
```

## What it demonstrates

* Filtering VET transfers by a block range
* Pagination of VET transfers from Thor response
* Filtering VET transfers by receipient address
